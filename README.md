# MVC Web Registration Application

A Spring MVC web application for user registration built using 
the Model-View-Controller architecture pattern.

## Tech Stack
- Java
- Spring MVC
- HTML
- Apache Tomcat

## Features
- User registration form
- MVC layered architecture
- Controller handles request/response
- Model binds form data

## How to Run
1. Clone the repository
2. Deploy on Apache Tomcat server
3. Access via `http://localhost:8080`
